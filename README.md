# eror-reporting-system-vanila js-basic
sistem sederhana untuk mencatat dan melacak laporan eror pada tim IT support
Error Reporting System (Basic)
Aplikasi sederhana untuk membantu tim dalam mencatat, melacak, dan mengelola laporan error secara terstruktur.
Project ini dibuat sebagai solusi awal untuk menghindari pencatatan error yang tidak terdokumentasi dengan baik (misalnya lewat chat atau catatan manual).


Fitur Utama
📝 Input laporan error (judul, deskripsi, prioritas)
📋 Menampilkan daftar laporan error
❌ Menghapus laporan

Tech Stack
HTML
CSS
Javascript (Vanilla JS)

Latar Belakang

Dalam proses pengembangan aplikasi, sering terjadi error yang tidak terdokumentasi dengan baik karena hanya disampaikan melalui chat atau komunikasi lisan.

Hal ini menyebabkan:

Informasi error mudah hilang
Sulit melakukan tracking
Tidak ada prioritas penanganan

Aplikasi ini dibuat untuk memberikan solusi sederhana agar tim dapat mencatat error secara lebih terstruktur.

Yang Saya Pelajari
Manipulasi DOM menggunakan JavaScript
Event handling (submit, click)
